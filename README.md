## Instructions

1. Add `packwiz-installer-bootstrap.jar` and `packwiz-installer.jar` in your server folder, same level as `mods` folder
2. Before staring up server, run following command in console

   ```
   java -jar packwiz-installer-bootstrap.jar -g -s server https://duskscorpio.github.io/NubTech-Mods/[NubTechCMP1 or NubTechCMP2]/pack.toml
   ```
